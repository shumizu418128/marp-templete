# marp-templete
marpでプレゼン作るときのテンプレファイル

## marp.md (マークダウンファイル)
- ある程度説明が書いてある
- このファイルをプレゼンに変換する

## marp.css (CSSファイル)
- 適当に装飾してある
- Marp for VSCode でCSSを使う際は、`marp.md` に書いてある設定をすること

## marp.pdf (PDFファイル)
- `marp.md` を出力するとこうなる

[Marp for VSCode](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode)
